# p2p-lending-credit-scoring-random-forest
Automating the credit verification process for a P2P lending platform using Random Forest Classifier to reduce manual assessment time and human error.

PT. DQ is a peer-to-peer (P2P) lending company that has recently launched a digital loan application feature. However, the credit verification process was still heavily manual, leading to bottleneck processing times and an increased risk of underwriting errors. To solve this business challenge, this project implements a Machine Learning approach using the **Random Forest Classifier** to automate the credit risk assessment. By utilizing a data-driven scoring system, the company can efficiently evaluate and predict whether a prospective borrower is eligible for a loan.

---

## 💼 Business Problem

* Inefficiency: Manual verification slows down the loan approval process, especially during high-demand periods.

* Human Error: Manual judgment introduces inconsistency and potential bias in risk assessment.

* Scalability: The business needs an automated, scalable solution to process numerous applications simultaneously without compromising accuracy.

---

## 🛠️ Tech Stack & Methods

* Language: Python

* Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn *(Sesuaikan dengan yang kamu pakai)*

* Algorithm: Random Forest Classifier

* Process Flow: Data Preprocessing -> Exploratory Data Analysis (EDA) -> Feature Engineering -> Model Training -> Sample Testing

---

## 📊 Key Insights & Results

* Successfully built a Random Forest model to classify borrower risk profiles.

* Tested the model on sample prospective borrowers to ensure fast and accurate decision-making.

* **Impact:** Transitioning to automated scoring cuts down verification time from days to mere seconds, significantly minimizing manual assessment overhead.
